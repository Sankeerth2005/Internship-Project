# Business Discovery & Profiles: Vocal for Sanatan
Version 1.0.0 • Search Discovery Layouts, Cards Specs & Profile Blueprints

This document outlines the visual layouts, card specs, and details structure that make up the business discovery experience in Vocal for Sanatan.

---

## 1. Redesigned Home Screen Layout

The home screen is re-imagined as an active discovery feed rather than a static list:

1.  **Warm Header Block:** Personal greeting (e.g. *"Namaste, Aditi!"* or *"Welcome back"*), location display drop-down selector, and notification bell icon.
2.  **Unified Search Bar:** Expandable input field supporting voice queries and autocomplete dropdowns.
3.  **Quick Actions Grid:** Circular shortcut buttons: `[ Category Search ]`, `[ AI Assistant ]`, `[ Nearby Map ]`, `[ Saved Stores ]`.
4.  **Seasonal Recommendations Banner:** Card highlighting seasonal needs (e.g., *"Festive Sweets & Gift Shops in Varanasi"*).
5.  **Dynamic Discovery Carousels:** Responsive horizontal listing blocks:
    *   *Recommended For You:* Personalized matches based on category preferences.
    *   *Nearby Businesses:* Ordered strictly by geolocated distance (under 2km).
    *   *Trending / Highly Rated:* High engagement listings showing 4.5+ star ratings.
    *   *Open Now:* Automatically filters and displays open storefronts based on local device time.
    *   *Recently Contacted / Viewed:* Last 3 visited or dialed businesses for fast repeat access.

## 2. Business Card Specification

Every merchant listing card renders these properties:

```
+-------------------------------------------------------+
|  [                     Store Image                 ]  |
|  [                      16:9 Aspect                ]  |
|                                                       |
|  Gupta Medicos [✓ Verified]                 (⭐ 4.8)  |
|  Vijay Gupta, Owner                                   |
|  Varanasi • Pharmacy • 400m away                      |
|                                                       |
|  +-------------------------------------------------+  |
|  |  [Call Owner]  |  [Directions]  |  (♥ Favorite) |  |
|  +-------------------------------------------------+  |
+-------------------------------------------------------+
```

*   **Store Image:** 16:9 aspect ratio thumbnail with `radius-lg` curve corners. Displays category fallback vector artwork if no photos are uploaded.
*   **Verified Badge:** A green checkmark badge next to the business title if approved by administrators.
*   **Owner Name:** Displayed directly under the business name to establish personal trust.
*   **Rating:** Stars + score average (e.g. `⭐ 4.8 (120 reviews)`).
*   **Distance & Category:** Dynamic walkability calculation and category details.
*   **Call & Directions CTA:** Quick action buttons that trigger native OS dialers and mapping apps.
*   **Favorite Heart:** Bounces using `curve-overshoot` and emits saffron confetti lines when favorited.

## 3. Premium Business Details Profile

Tapping a card opens the detailed profile view containing:

1.  **Media Carousel:** Horizontal gallery of store photos, including zoom controls and a "View All" modal.
2.  **Verification Banner:** Shows the owner's verification status, owner's profile picture, and the date they registered.
3.  **Owner Information & Story:** A short biography written by the merchant (e.g. *"Vijay Gupta founded this store in 1998 to serve the local community..."*).
4.  **Interactive Operating Hours Grid:** Displays current status (Open/Closed), weekly hours, and highlights **Peak Hours** (using simple bar charts to show busy times).
5.  **Facilities & Amenities Checklist:** Payment methods accepted (UPI, Cash, Cards), languages spoken (Hindi, English), parking availability, and wheelchair access.
6.  **AI Reviews Summary:** A bullet-point summary of user reviews generated by the AI (e.g. *"Customers praise Gupta Medicos for fast service, friendly behavior, and reliable stock"*).
7.  **Interactive Map Dock:** A MapLibre mini-view showing the storefront coordinates, containing a prominent "Get Directions" primary button.
8.  **Ratings & Reviews Board:** Individual user reviews, sorting selectors, and a reply field for verified owners.
9.  **Related Businesses Carousel:** Recommends alternative stores in the same category located nearby.
